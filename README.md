# MilkyWayExplorer
🚀 Devlog – MilkyWayExplorer (Open-Source Sandbox MMORPG)
Salut !
Je travaille actuellement sur MilkyWayExplorer, un MMORPG sandbox cross-platform (Linux / Windows) inspiré par World of Warcraft, Minecraft et Boundless.
Le jeu utilise un moteur 3D entièrement développé en C++ et se concentre sur :
🌍 Open world procédural
🪐 planètes générées procéduralement
🏗 construction voxel
⚔ PvP / PvE
👥 multijoueur massif
💰 économie dynamique
🏰 factions / teams
🗝 quêtes et donjons
Lore du mode solo :
Vous incarnez un astronaute perdu dans l'espace qui doit survivre, explorer différentes planètes et trouver un moyen de rentrer sur Terre.
Chaque planète aura son propre environnement :
    • désertique
    • habitable
    • hostile
    • difficulté variable
Le jeu sera entièrement moddable via une API Lua (SOL2).
Un éditeur voxel art est aussi en cours de développement pour faciliter la création de contenu.
Le projet est en développement actif et une version Alpha pour les premiers tests arrive bientôt.
Je recherche aussi des personnes intéressées pour contribuer :
    • musique
    • sound design
    • voxel art
    • idées de gameplay
Si ça vous intéresse, je partagerai bientôt plus d’infos et le repo GitHub !
